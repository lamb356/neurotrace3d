# NeuroTrace3D

Web-based 3D neuron morphology viewer/editor.

## Packages

- `@neurotrace/swc-parser` — SWC file parser and serializer
- `@neurotrace/neuron-tree` — Neuron tree data structure (stub)

## Apps

- `web` — Web application (stub)

## Development

```bash
pnpm install
pnpm build
pnpm test
pnpm typecheck
```
